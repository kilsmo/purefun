# Operators

• All operators are **binary**
• All operators are **infix**
• All operators are **left-associative**

• Operators have fixed precedence levels (higher binds tighter):

* ```Level 6: ( )     grouping```

* ```Level 5: * / %   multiplicative```

* ```Level 4: + -     additive```

* ```Level 3: == !=   equality```

* ```Level 2: &&      logical AND```

* ```Level 1: ||      logical OR```
