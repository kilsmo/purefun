Purefun supports 5 types of literals:

* int      — an integer of any size, e.g., 23, -14, 0, 12345678901234567890
* num      — a floating-point number, e.g., 47.725
* bool     — a boolean, either true or false
* string   — an immutable string, defined with single quotes: 'hello' (preferred), or double quotes: "World" if you want the actual string to be World
* char     — a single Unicode character, e.g., 'a', 'Ω'
